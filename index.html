HTML

  • Enter your HTML notes here
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element typically contains address details or copywrite information.
  • src allows you to attach an image source but make sure to give it an alt for accessibility and search engine optimisation(SEO).
  • Headers go from h1 to h6 and represent the tiers of heading with h1 being the highest and h6 the lowest.

CSS

  • Enter your CSS notes here
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • use classes to asign design elements in the external style sheet, this is done by using .'classname'
  • Within css you use property: value to set up design elements eg: color: blue;
  • You also end these with a ; before moving onto the next line to show that line of css is complete

Git
